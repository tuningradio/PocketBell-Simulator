=================================
PocketBell Simulator Ver 1.0 by JA1XPM 2026/04/16
=================================

PocketBell GUI 仕様書

1. 概要
本プログラムは、アマチュア無線の FM 音声回線上で DTMF を使って
ポケベル風のメッセージ送受信を行う GUI アプリケーションである。
PC 上にポケベル風画面を表示し、宛先と本文を DTMF 電文へ変換して送信し、
受信した DTMF 電文を復号して表示する。

2. 構成ファイル
- index.html
  GUI 本体
- style.css
  GUI レイアウトとホットスポット定義
- app.js
  画面描画、入力処理、履歴管理、WebSocket 通信
- pager_tx_bridge.py
  GUI と DTMF 送受信処理の橋渡し
- dtmftest_pager.py
  DTMF の送出、受信判定、COM ポート DTR 制御

3. 通信電文形式
送信電文は以下の形式とする。

*2*2[宛先CALL]A0A[送信者CALL]A0A[本文データ]##

- 宛先CALL
  英大文字 3 桁コードと数字 1 桁コードの組み合わせ
- 送信者CALL
  SET で設定した自局コールサイン
- 本文データ
  *2*2 フリーワード本文または *4*4 定型文本文を格納
- 終端
  ##

4. コールサイン
- 英字は以下の 3 桁コードを使用する
  A=111 B=112 C=113 D=114 E=115
  F=121 G=122 H=123 I=124 J=125
  K=131 L=132 M=133 N=134 O=135
  P=141 Q=142 R=143 S=144 T=145
  U=151 V=152 W=153 X=154 Y=155 Z=156
- 数字は 0 から 9 をそのまま 1 桁で使用する
- 受信側は JA8AA、JH1HUW、JA1XPM、7K1ABC、8N1YOKO のような
  日本の一般的なコールサインを復号できる

5. 本文仕様
本文は、利用者が本文先頭にコマンドを含めて入力する。

5-1. フリーワード本文
- 入力形式
  宛先,*2*2カナ本文
- 利用者はカナをそのまま入力する
- 送信時に内部でフリーワードテーブルへ変換する
- 受信表示はカナで表示する

5-2. 定型文本文
- 入力形式
  宛先,*4*4NN
- NN は 2 桁の定型文番号
- 受信時は定型文テーブルで文言へ展開して表示する

6. 宛先入力
- 入力欄は 宛先,本文 の形式とする
- 宛先を省略した場合は CQ 扱いとする
- 一度に送れるメッセージは24文字までです。
- 例
  CQ,*2*2キョウノミニイクカ
  JH1HUW,*4*402

7. 表示仕様
7-1. 右上ステータス
- 自局コールサインは常に右端に表示する
- 最新送信時
  宛先 < 自局CALL
- 最新受信時
  送信元 > 自局CALL

7-2. 左上表示
- 月日表示
- 時刻表示
- 新着受信時は左端に ※ を表示
- ※ は新着受信から 5 分で消灯する

7-3. 本文履歴表示
- 2 行表示
- 送信時
  ←宛先CALL 本文
- 受信時
  →送信元CALL 本文
- 表示件数は最大 10 行
- 上下ボタンで履歴スクロール可能
- 履歴は localStorage に保存し、再起動後も復元する

8. SET 機能
- SET ボタンで自局コールサイン編集画面を開く
- SAVE で my_call を保存する
- 保存先は localStorage
- 保存後はバックエンドへも送信し、受信判定に反映する

9. 受信表示条件
- 宛先が自局コールサインと一致した場合のみ表示する
- 宛先が CQ の場合は表示する
- それ以外の宛先は受信しても表示しない

10. オーディオと PTT
- PC オーディオ出力を無線機入力へ接続する
- 無線機音声出力を PC オーディオ入力へ接続する
- COM ポート DTR を PTT 制御に使用する

11. 起動時設定
start_tx_gui.bat で以下を設定する。

set OUTDEV=送信用出力デバイス番号
set INDEV=受信用入力デバイス番号
set COM=使用COMポート

VOX(Signalinkを含む)を使用する場合は以下を設定する。

set COM=NONE

デバイス番号確認コマンド
py dtmftest_pager.py devices

12. 動作の流れ
送信時
- Enter
- DTR ON
- DTMF 送出
- DTR OFF

受信時
- 音声入力から DTMF を監視
- ## を検出すると 1 文確定
- 自局宛または CQ 宛なら復号して表示
