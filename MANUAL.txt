=================================
PocketBell Simulator Ver 1.0 by JA1XPM 2026/04/16
=================================

PocketBell GUI 取扱説明書

1. インストール
(1) Python
    Python 3.10 系を使用する。
    `py -3.10` が使える状態を推奨する。

(2) ライブラリ
    本プログラムの動作に必要なライブラリは requirements.txt にまとめてある。
    以下を実行して導入する。

    py -3.10 -m pip install -r requirements.txt

    `py` が使えない環境では、使用中の Python に合わせて

    python -m pip install -r requirements.txt

    でもよい。

(3) 配線
    PC オーディオ出力を無線機入力へ接続する。
    無線機音声出力を PC オーディオ入力へ接続する。
    COM ポートを PTT 制御線へ接続する。

(4) デバイス番号確認
    次のコマンドでオーディオデバイス番号一覧を表示する。

    py dtmftest_pager.py devices

(5) start_tx_gui.bat の設定
    起動前に以下を設定する。

    set OUTDEV=送信用出力デバイス番号
    set INDEV=受信用入力デバイス番号
    set COM=使用COMポート

2. 起動
start_tx_gui.bat を実行する。

3. 自局コールサイン設定
(1) 画面上の SET ボタンをクリックする
(2) 入力欄に自局コールサインを入力する
(3) SAVE をクリックする
(4) 右上表示と受信判定に反映される

4. 送信
入力欄には 宛先,本文 の形式で入力する。

例
CQ,*2*2キンキュウ
JH1HUW,*4*402

Enter で送信する。

本文の先頭には利用者がコマンドを直接入力する。

- *2*2
  フリーワード本文
- *4*4NN
  定型文番号 NN

宛先を省略した場合は CQ 扱いになる。

5. 受信
受信音声から DTMF を監視し、## を受信すると 1 文として確定する。
宛先が自局コールサインまたは CQ の場合のみ画面へ表示する。

6. 画面表示
- 左上
  月日、時刻、新着受信時の ※
- 右上
  最新の相手先表示
  送信時は 宛先 < 自局CALL
  受信時は 送信元 > 自局CALL
- 本文欄
  2 行表示
  送信時は ←宛先CALL 本文
  受信時は →送信元CALL 本文

7. 履歴
- 履歴は最大 10 行保存する
- 上下ボタンでスクロールできる
- 履歴は localStorage に保存され、再起動後も復元される
- 新着受信マーク ※ は受信後 5 分で消える

8. 注意
- 本プログラムは良好な FM 音声回線での利用を前提とする
- 通信補助や再送制御は持たない
- 本文は *2*2 フリーワードまたは *4*4 定型文として扱う
